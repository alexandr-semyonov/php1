1. var_dump (1 == 1.0);
Нестрогое сравнение без учета типов. Число с плавающей точкой будет преобразовано в целое.

2. var_dump (1 === 1.0);
Строгое сравнение с учетом типов. Преобразования типов не происходит, поэтому целое число 1 не равно числу с плавающей точкой 1.0.

3. var_dump ('02' == 2);
Нестрогое сравнение без учета типов. Строка будет преобразована в число 2 и сравниваться будут числа.

4. var_dump ('02' === 2);
Строгое сравнение с учетом типов. Преобразования типов не происходит, поэтому строка 02 не равна целому числу 2

5. var_dump ('02' == '2');
Нестрогое сравнение без учета типов. Каждая строка будет преобразована в число и сравниваться будут как числа.
